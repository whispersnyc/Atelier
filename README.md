# Atelier: Modding Studio

### Installation
1. Install [Microsoft .NET Runtime 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-8.0.28-windows-x64-installer)
2. Download latest version from [Releases](https://github.com/clownfetus/Atelier/releases)
3. Extract contents of downloaded .zip
2. Run Atelier.exe
