# Atelier: Modding Studio

### Installation
1. Install [Microsoft .NET Runtime 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-8.0.28-windows-x64-installer)
2. Download latest version from [Releases](https://github.com/clownfetus/Atelier/releases)
3. Put the .exe in an empty folder
2. Download the following tools and put it in a `Tools\` folder next to the .exe
    - [Tools\retoc-rivals-cli\retoc-rivals-cli.exe](https://github.com/natimerry/repak-rivals/releases/download/v3.4.1/retoc-rivals-cli-x86_64-pc-windows-msvc-self-contained.zip) (extract)  
    - [Tools\retoc-rivals-cli\oo2core_9_win64.dll](https://github.com/natimerry/repak-rivals/blob/master/oo2core_9_win64.dll)   
    - [Tools\UAssetTool.exe](https://github.com/XzantGaming/UAssetToolRivals/releases/download/v1.5.5/UAssetTool-win-x64.zip) (extract)
    - [Tools\Mappings\Marvel_S8.5.usmap](https://spacedepot.github.io/SpaceHost/) (rename)  
    - [Tools\AES_KEY.txt](https://discord.com/channels/1419106202511609958/1485413590310584374/1485417747834863616) (copy-paste key into a txt file)  
    - [Tools\MarvelRivalsCharacterIDs.md](https://raw.githubusercontent.com/donutman07/MarvelRivalsCharacterIDs/refs/heads/main/MarvelRivalsCharacterIDs.md) (keep this updated for new skins)
